
# Foma

Easy form builder inspired by Notion's UI

## Installation

Clone the project into your local workspace and create a `.env.local` file and replace with your HarperDB variables.
```bash
VITE_HARPERDB_URL=
VITE_HARPERDB_TOKEN=
```

## Usage
Run these commands to start the development server.

```bash
> npm install
> npm run dev
```
